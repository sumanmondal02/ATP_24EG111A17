# Frontend Examples and Assignments (26-03-14_16_17)

This folder is a curated collection of small frontend projects, exercises and UI samples created to practice HTML and CSS fundamentals and to explore layout techniques and styling options.

## What you'll find here
- A variety of single-page examples demonstrating common frontend patterns:
  - `Employee_Card-1/`, `Employee_Card-2/`, `Employee-Cards/` — card-style profile components.
  - `Employee_Cards-2_tailwind/`, `CompanyWebsite-1_tailwind/`, `Website_assignment_tailwind/` — Tailwind CSS based examples (includes `src/` with `input.css`/`output.css`).
  - `Form-1/` — form layout and custom input styling.
  - `Grid-1/`, `Table-1/`, `Table-2_RowColSpan/` — grid and table layouts with examples of rowspan/colspan.
  - `Position-1/`, `MediaQuery-1/` — CSS positioning and responsive examples.
  - `QuickBlog_Home-Frontend/`, `HotelPortfolio_assignment/`, `HotelPortfolio2/`, `UserPortfolio_assignment1/` — small site/homepage templates.
  - `Cities_LinkHTML/` — an example showing linked pages and a small multi-page site with `cities.html`, `bangalore.html`, `chennai.html`, `hyd.html` and stylesheet `style.css`.
  - `Font1/`, `Font2-GoogleFont/` — examples using custom and Google fonts.
  - `Whatsapp-Test/` and various `Project-*` directories — assorted layouts and project trials.
  - `assets/` — images and other static assets used across examples.

## What this teaches
- Core HTML structure: semantic tags, content grouping, and page composition.
- CSS fundamentals: box model, typography, colors, borders, shadows, and gradients.
- Layout techniques: floats, CSS Grid, Flexbox, and manual positioning.
- Responsive design basics: media queries and adapting layouts for different screen sizes.
- Component styling: cards, forms, tables, and navigation headers.
- Working with images, background images, and asset organization.
- Tailwind CSS basics (where included) and workflow for generating `output.css`.
