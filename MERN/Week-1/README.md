# JavaScript Exercises — 26-02-18 & 19

This folder contains small JavaScript practice files that demonstrate basic language features, simple algorithmic problems, arrays, objects, and function topics.

Files
- `Biggerof2Numbers.js` — Find the larger of two numbers.
- `Biggerof3Numbers.js` — Find the largest of three numbers.
- `SumofMarksArray.js` — Sum elements of an array.
- `SmallestElementArray.js` — Find the smallest element in an array.
- `FunctionReceive3NumberArgsReturnBig.js` — Function that returns the largest of three args.
- `FunctionReceiveArgArray.js` — Function that returns the sum of an array argument.
- `FunctionArraySearch.js` — Function to search an element in an array and return its index.
- `BasicOperationsObjects.js` — insert/remove/update operations on an employees array.
