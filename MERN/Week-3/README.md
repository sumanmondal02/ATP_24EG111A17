# Backend & Async Exercises — 26-03-05, 06 & 07

This folder contains backend-focused examples and asynchronous JavaScript exercises from the week.

Overview
- `Backend-3/` — a small Express-based demo with modular APIs, models, and middleware (`verifyToken`).
- `Backend-3 copy/` — alternate demo that includes an `auth/` helper and the same modular structure for comparison.

What this folder is for
- Learn how to wire a Node/Express backend with modular routes, models, and middleware.
- See examples of authentication middleware (`verifyToken`) and how models are organized (`models/`).
- Practice asynchronous patterns (`async`/`await`) and how they appear in real server handlers.
