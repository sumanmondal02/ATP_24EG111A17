# JavaScript Exercises — 26-02-25, 26 & 28

This folder contains practice exercises that demonstrate array and object operations using common higher-order methods (`filter`, `map`, `reduce`, `find`, `findIndex`), intermediate JavaScript language features and common patterns:
- Classes and instance methods
- Constructors and object-oriented basics
- Private members and inheritance
- Optional chaining, nullish coalescing
- Spread/rest, destructuring
- Structured cloning (deep copy)
- Asynchronous JS: timers, promises, simulation
- Small practical examples (library, payroll-like behaviors, exam/OTP simulators).

and introductory backend examples and two small demo projects that illustrate how to build simple Node.js servers and modular APIs.


Files
- `DailyTemperatureAnalyzer.js` — temperature array operations (filter/map/reduce/find).
- `ShoppingCartSumm.js` — shopping-cart summary (filter/map/reduce/find).
- `OnlineCourseDisplay.js` — course-name processing and display.
- `StudentPerformance.js` — grading and statistics for student marks.
- `StudentMarksList.js` — mark-level helpers (pass, highest, etc.).
- `EmployeePayroll.js` — payroll computations and net salary.
- `MovieStreamingPlatform.js` — movie list processing and averages.
- `BankTransactionAnalyzer.js` — transaction filtering and balance calculation.
- `TaskManagementSystem/` — small task app (separate mini-project).
- `Backend-1/` — minimal server example (package.json, server.js, and `req.http` with sample requests).
- `Backend-2/` — extended example with modular APIs under `APIs/` (includes `ProductAPI.js` and `UserAPI.js`), plus server and request samples.
