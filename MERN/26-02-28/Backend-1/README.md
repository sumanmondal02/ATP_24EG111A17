### Steps to create backend 

1. Generate Package.json file (package.json it is the main file without it backend wont work)
    - npm init -y (init stands for initialize)
    ### "type": "module" keep this 

2. Create HTTP Server 
    - Install Express Module
        - npm install express
        - npm i express ### or you can use this 
    - Import express module

### install "npm install -g nodemon" so that the server is automatically running no need of restarting server after every update
### -g stands for global and also use nodeman instead of using node
